Variable code

The Path standard module.

This module is used to manipulate paths which are mostly strings.

Although more functions like changing directory are provided, some terminal commands could be used. Its also here that valid path names are validated like the filename, and directory name.

Notice.

Notice that Windows allows both \(backslash) and /(slash) to be path separators, variable code prefers /(slash) for it will make your code cross-platform without a need to change the separators.
