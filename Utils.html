Overview

The Utils standard module.

The Utils module is automatically imported in the vC interpreter when it starts both in interactive or execution sessions similar to the Python sys module. Some languages define Objects from such a module, this one is similar to theme and it also changes datatypes.

Since this is an essential module, its updates are regular mostly concerning stability and efficiency.
