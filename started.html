Get started with Variable code

Variable code or vC is a general purpose interpreted programming language aiming to provide the suitable entry powerful point for programming beginners. Speed is greatly varied, syntax is very readable based on ideas from developers' imaginary programming languages.

Much similar to any other programming languages, get the executable and then enjoy the language!

Learning Variable code.

Developer speed here is improved to maximum meaning the syntax is very easy to pick up, easy as python flowing like javascript with weak typing thats good at most points.

Guessing an attribute from one of your favourite programming languages could work here since concepts are the same and there is less easier we could provide than the existing.

Expect to build.

Vc (Variable code) is not yet at a stable version, much is being developed as for now but all new features will be noted on this website. Checkout the News page.

Never the less the language has a bunch of modules that inteface with the operating system including.

  • IO operations: Writing files is very possible and kinder stable.
  • Socket networking: The socket API is provided too although not to the stable version so expect some bugs although not guranteed they will ocurr.
  • The list is endless check out the modules page and some of our tutorials explaining the usage of the modules.
