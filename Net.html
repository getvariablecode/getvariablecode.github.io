The Net standard module.

The vC Net module uses libcurl to provide networking functionality in the language. It is used to fetch URLs (Uniform Resource Locators). It provides functions to fetch URLs using a variety of different protocols supported by libcurl.

Module is still in development, there is no full support for synchronously and asynchronously data fetching from a network.
