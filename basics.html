5. loop

These change the control flow of the program.

.1 while

Bases on a condition to loop its scoped code, if the condition is not satisfied, then the loop will break. You can even break manually with the break key.

count = 0

while (count < 100) {
	print(count)
	count+=1
}

print('Loop is done running.')

.2 for

Iterates iterable data until the last item returning it to the assigned identier that can be accessed in the scoped code.

letters = ['a', 'b', 'c', 'd']

for i in letters {
	print(i)
}
