Overview

The Socket standard module.

The variable code Socket module creates network sockets with the Berkeley socket interface. The Socket module provides various objects, functions and related exceptions for building fully functioning network applications including client and server programs.

Notice that some your knowledge like Protocols numbers would apply in the your code as you use this module. Since in most networking cases the numbers are the same that present different protocols, socketTypes etc.

Trick, you can easily build your server with Variable code and a client with any programming language that supports sockets or even use the browser in this, these languages can be able to communicate. So you may raise a Browser-based application with this language Or use it as a backend language with just the sockets module although it may not be efficient for now.

Module is still in development but it works.

# hostname

hostname! is a variable that already holds the hostname of the machine the interpreter is running on.

# AF (addressFamily)

This is an object having addressFamilies that are supported by the vC sockets module in the new_socket instance object.

INET INET6 UNIX IPX UNSPEC APPLETALK

# ST (socketType)

This is an object having socketTypes that are also supported by the vC sockets module in the new_socket instance object.

RDM RAW DGRAM STREAM SEQPACKET

# SD (howToShutdown)

This is an object having the ways the socket should shutdown the connections by disabling its capabilities, they are also numbers ranging from 0 to 2 disabling socket connection power either SENDING, RECEIVING or BOTH.

SEND RECV BOTH

# lastsocketError

This function retrives the last occurred error on its time of calling from the operating system, cuts across other sockets created by the OS whether by another language or by OS itself.

string lastsocketError()

# lastblockedFail

Checks if the last called socket connection failed and yet it was blocking or if it reached the set timeout without connecting.

bool lastblockedFail()
