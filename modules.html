Overview

Variable code builtin library

Variable code builtin modules are built in the language, there is no need to install them via a package manager, just use the import key work to add theme to your code.

These modules are optimized, they extend the language's features, they target speed and performance and some are written entirely in Dlang and C.

These cover a wide topic, they increase the learning curve so its better to learn them via a tutorial.

Modules

Simple description

File

Handles file operations and io streams like writing to files, deleting and finding file sizes.

Path

Any code concerning path manuplation and folder handling is handled here. Provides other functions like Unix cmds.

Thread

Creates threads to run processes concurrently or at the same time and also has functions to deal with the created threads.

Time

Time telling, sleep function and formating to desired output is done by this module.

Sockets

Provides functions for low level programming like creation of TPC, UDP etc connects to handle network operations.

Math

A simple math library with all mathematical functionality you may need. Like cos, tan etc.

Random

Genrates random numbers, excuvates random items from arrays, shuffles arrays and more functionality will be added.

Net

Uses the libcurl library to retrive and request content from internet. More similar libraries are to be provided.

Environ

Manuplates the vC interpreter variables like module path and commandline arguments.

More libraries

More libraries or modules if not listed here are custom, they are developed by other developers. So there documentation is expected to be provided by them on there sites or tutorials.

You can install these modules with a package manager although ours is in development. So download the package and include in your working space or checkout more about vC configuration.
